3.To check current condition:

4.To check corrections:

5.To back from stage(undo add .):
git reset 'NAME'

6.To undo all add .:
git reset

7.To undo changes in a file:
git chechout 'NAME'

8.To undo all changes in all files:
git checkout .

9.To upload to stage position:
i>      git add -A  [to stage all]
ii>     git add .   [to stage new, modified files, without deleted ones]
iii>    git add -u  [to stage deleted, modified files, without new ones]

10.To pull changes from github:
git pull origin master

11.To push changes to github:
git push -u origin master

12.To copy/clone folder from github to a directory:
git clone ssh_name 

13.To copy/clone repository without making a folder:
git clone ssh_name .
